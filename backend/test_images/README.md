# 测试图片目录

## 使用说明

将题目图片（.jpg 或 .png 格式）放入此目录，用于测试阿里云教育 OCR 识别效果。

## 图片要求

- **格式**：JPG、PNG
- **内容**：印刷体题目文字（可包含数学公式）
- **清晰度**：确保文字清晰可读
- **数量**：建议放入 5-10 张不同类型的题目图片

## 测试题目类型建议

1. **数学题**：包含数学公式（如一元一次方程、几何题）
2. **文字题**：纯文字题目（如语文阅读理解）
3. **应用题**：实际场景题目（如物理、化学实验题）
4. **选择题**：包含选项的题目
5. **解答题**：需要详细解答的题目

## 运行测试

```bash
# 在 backend 目录下运行
cd /Users/pinya_hu/Desktop/tare/AI_quiz_new/backend

# 运行测试脚本
python scripts/test_aliyun_ocr.py
```

## 验收标准

测试脚本会自动验证以下指标：

- **识别成功率**：≥90%
- **平均置信度**：≥90%
- **平均耗时**：≤3 秒
