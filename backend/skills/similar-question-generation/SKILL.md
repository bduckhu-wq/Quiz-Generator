# 相似题生成技能

## Overview

基于原题生成保持核心学习目标但表层维度变化的相似题。通过四层可控模型精确控制相似度，统一采用70%相似度设计。

**核心理念：相似度是比率，题目体量越大，维持同样相似比需要改变的部分越多。**

**违反规则的字面意思就是违反规则的精神。不存在"形式上遵守但实质上变通"的说法。**

## Four-Layer Control Model

### 核心层（锁死 = 0%变化）

**MUST NOT改变以下任何一项，改变任何一项即视为改编题而非相似题：**

| 变量 | 说明 |
|------|------|
| 最小知识点 | 精准到考点，如「勾股数定义」 |
| 核心数学模型 | 公式、等量关系、几何拓扑 |
| 解题逻辑链 | 步骤顺序、突破口 |
| 能力考查层级 | 理解/应用/综合 |
| 设问核心目标 | 列方程/求值/判对错 |

### 结构层（≤10%变化）

**MUST至少改变1处：**

| 维度 | 说明 |
|------|------|
| 题型 | 客观题/主观题/实操题 |
| 干扰项类型 | 概念混淆/计算错误/逻辑颠倒/范围遗漏 |
| 求解对象 | 基础量/中间量/参数/关系判定 |
| 设问角度 | 正向/反向/逆向验证/存在性/最值 |
| 设问层级 | 单问/两问递进/多问阶梯/开放探究 |
| 条件呈现 | 显性/间接推导/图表/隐藏条件 |

### 表达层（30-50%变化）

**MUST完全陌生化场景（背景/场景/人物三要素绑定变化，只记1处）：**

| 维度 | 说明 |
|------|------|
| 信息载体 | 纯文字/几何图/函数图像/表格/图文混合 |
| 语言风格 | 严谨书面/生活口语/故事风 |
| 信息顺序 | 顺叙/问题前置/条件拆分 |

### 数值层（80-100%变化）

| 维度 | 说明 |
|------|------|
| 具体数值 | MUST换数字 |
| 数量级 | 百/千/万，可大幅改动 |
| 单位 | 涉及单位换算才算变化 |

## 70%相似度规则

**MUST同时满足以下所有条件，缺一不可：**

1. **场景MUST完全陌生化**（不能只换人名/地名）
2. **结构层MUST至少改变1处**
3. **选项结构MUST变化**（顺序、位置、表述）
4. **干扰项MUST来自模型的不同误解角度**

## 体量与变化量

| 体量 | 问数 | MUST改动的最少处数 |
|------|------|---------------------|
| 小题 | 1问 | 2处（跨2维度） |
| 少问题 | 2-3问 | 3-4处（跨3维度） |
| 中等题 | 3-5问 | 4-6处（跨3维度） |
| 大题 | 5问以上 | 6处以上（3维度全动） |

**核心层永远不动。体量只影响结构层/表达层/数值层的改动量。**

## 干扰项设计原则

**MUST遵守：所有干扰项必须来自对核心模型的真实误解。**

**FORBIDDEN：硬塞无关约束作为干扰项。**

| ✅ MUST | ❌ MUST NOT |
|---------|-------------|
| 来自模型的真实误解 | 硬塞无关约束 |
| 四选项对应真实错误类型 | 加无关附加条件 |

**判断标准：这个干扰项对应的错误，是学生真实会犯的吗？跟本道题要考察的核心能力有关吗？**

## 输出格式

**MUST严格输出以下JSON格式，MUST NOT输出思考过程：**

```json
{
  "question": "题目内容，含选项",
  "answer": "正确答案",
  "explanation": "详细解析"
}
```

## Red Flags - 出现以下情况说明生成有误，MUST重新生成

- 只改了数值和人物名称，结构层未动
- 场景与原题相似（只是换了人名/地名）
- 干扰项包含与核心模型无关的约束条件
- 题目条件不足以支撑唯一答案
- 计算量超出常规考试要求

## 常见借口与纠正

| 借口 | 现实 |
|------|------|
| "原题场景已经很好了，不需要陌生化" | 场景陌生化是硬性要求，不是可选项 |
| "只改数值也算结构层变化" | 改数值属于数值层，不属于结构层 |
| "干扰项加个约束更严谨" | 无关约束会误导考查方向，MUST来自真实误解 |
| "这道题体量小，改1处就够了" | 小题MUST改2处，这是最低要求 |

## ❌ 负面示例（严禁模仿）

### 错误案例：相似度95%（不合格）

**原题**：
```
已知椭圆 C：x²/a² + y²/b² = 1 (a>b>0) 的离心率 e=2√2/3，
以椭圆C的四个顶点为顶点的四边形面积为 6.
(1) 求椭圆C的方程；
(2) 设 A 为椭圆 C 的上顶点，P 为椭圆上任意一点，求 |AP| 的最大值。
```

**❌ 错误的相似题（只改了数字）**：
```
题1：已知椭圆 C：x²/a² + y²/b² = 1 (a>b>0) 的离心率 e=1/2，
     以椭圆C的四个顶点为顶点的四边形面积为 4√3.
     (1) 求椭圆C的方程；
     (2) 设 A 为椭圆 C 的右顶点，P 为椭圆上任意一点，求 |AP| 的最大值。

题2：已知椭圆 C：x²/a² + y²/b² = 1 (a>b>0) 的离心率 e=√3/2，
     以椭圆C的四个顶点为顶点的四边形面积为 4.
     (1) 求椭圆C的方程；
     (2) 设 A 为椭圆 C 的左顶点，P 为椭圆上任意一点，求 |AP| 的最大值。

题3：已知椭圆 C：x²/a² + y²/b² = 1 (a>b>0) 的离心率 e=√6/3，
     以椭圆C的四个顶点为顶点的四边形面积为 4√3.
     (1) 求椭圆C的方程；
     (2) 设 A 为椭圆 C 的下顶点，P 为椭圆上任意一点，求 |AP| 的最大值。
```

**❌ 问题分析**：
- 题干结构完全一致（只改了数字）
- 第(1)问完全相同（都是"求椭圆方程"）
- 第(2)问完全平行（都是"求|AP|最大值"，只改了顶点位置）
- 表达层未变化（条件呈现方式完全一致）
- 结构层未变化（设问角度完全一致）
- **实际相似度：95%**

## ✅ 正面示例（应该模仿）

**同样的原题，正确的生成方式（相似度70%）**：

```
题1：已知椭圆 C：x²/a² + y²/b² = 1 (a>b>0) 的离心率 e=1/2，
     以椭圆C的四个顶点为顶点的四边形面积为 8√3.
     (1) 求椭圆C的方程；
     (2) 设 O 为坐标原点，P 为椭圆上任意一点，求 |OP| 的最大值及此时点P的坐标。

题2：已知椭圆 C 的短轴长为 4，焦距为 2√3.
     (1) 求椭圆C的方程；
     (2) 设 F 为椭圆的右焦点，P 为椭圆上任意一点，求 |PF| 的取值范围。

题3：椭圆 C 的焦点在 x 轴上，离心率为 √3/2，且经过点 (2, 1).
     (1) 求椭圆C的标准方程；
     (2) 若直线 y=kx+1 与椭圆C交于A、B两点，求△AOB面积的最大值（O为原点）。
```

**✅ 改进点**：
- 题1：求解对象改变（上顶点A → **原点O**）
- 题2：条件呈现改变（离心率+面积 → **短轴长+焦距**），设问改变（最大值 → **取值范围**）
- 题3：条件呈现改变（完全不同的表述），设问改变（求最大值 → **求三角形面积最大值**）
- **实际相似度：70%**

## 关键原则

**如果生成的3道题，第(1)问和第(2)问的主体部分都一样，那一定是错的！**

- ❌ 错：(1)都是"求椭圆方程" (2)都是"求|AP|最大值"
- ✅ 对：(1)有不同 或 (2)有明显差异（不只是改字母）
