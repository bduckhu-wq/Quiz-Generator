```
用户输入出题需求
  ↓

  ↓

  ├─ 课后作业 → 执行"课后作业场景决策流程"

  └─ 考前复习 → 执行"考前复习场景决策流程"★核心
  ↓
生成参数确认面板
  ↓
用户确认 → 生成试卷
```

## 🔹 场景1：课后作业

### 决策流程

```
用户："帮我出点初二数学课后作业"
  ↓
AI 提取：学科=数学, 年级=初二, 场景=homework
  ↓
检查内容入口是否缺失
  ├─ 是 → 追问："作业内容？"
  │        选项：
  │        📕 按教材章节（如：第16章）
  │        💡 按知识点（如：二次根式化简）
  │   ↓
  │   用户选择："第16章"
  │   ↓
  │   检查教材版本是否缺失
  │   ├─ 是 → 追问："使用的教材版本？"
  │   └─ 否 → 跳过
  │   ↓
  │   内容入口 = {textbook: "人教版", chapter: "第16章"}
  │
  └─ 否（用户已指定内容）→ 跳过追问
  ↓
应用场景策略：
  - 难度分布 = {简单:50%, 中等:40%, 困难:10%}
  - 题型分布 = {选择:30%, 填空:30%, 解答:30%, 其他:10%}
  - 题量 = 10 题（默认）
  ↓
生成参数确认面板
```

```
     [人教版] [北师大版] [苏教版]
  🎯 场景：课后作业
  📊 难度：简单50%、中等40%、困难10%

  点击"确认生成"或继续调整参数~"
```

**查询逻辑伪代码**：

```python
def get_chapter_range(textbook: str, subject: str, grade: str, 
                     semester: str, exam_type: str) -> list[str]:
    """
    根据教材版本、学科、年级、学期、考试类型查询章节范围。
    
    Args:
        textbook: 教材版本（如"人教版"）
        subject: 学科（如"数学"）
        grade: 年级（如"初二"）
        semester: 学期（如"上学期"/"下学期"）
        exam_type: 考试类型（如"期中"/"期末"）
    
    Returns:
        章节列表（如["第16章", "第17章", ...]）
    """
    mapping_table = load_textbook_mapping()
    
    try:
        chapters = (
            mapping_table
            [textbook]
            [subject]
            [grade]
            [semester]
            [exam_type]
        )
        return chapters
    except KeyError:
        # 如果查询失败，返回空列表（降级为手动输入）
        logger.warning(f"未找到映射：{textbook}-{subject}-{grade}-{semester}-{exam_type}")
        return []
```

## 🛠️ 实现要点总结

### 1. 后端需要实现的数据表

| 数据表 | 用途 | 优先级 |
|-------|------|--------|
| **textbook_semester_mapping** | 教材版本-学期-章节映射 | P0（期中期末场景必须） |
| **knowledge_to_chapter** | 知识点→章节反向映射 | P1（单元测验场景优化） |
| **review_recommendation** | 复习场景智能推荐 | P1（复习场景优化） |

### 2. AI Prompt 需要新增的能力

| 能力 | 说明 | 实现方式 |
|-----|------|---------|
| **考试类型识别** | 识别"期中"/"期末"关键词 | NLU 关键词匹配 + 上下文推断 |
| **学期推断** | 根据时间/年级推断学期 | 规则引擎（如：9-12月 = 上学期） |
| **章节范围查询** | 查询教材映射表 | Tool Calling：`query_chapter_range()` |
| **知识点反向映射** | 知识点→章节 | Tool Calling：`map_knowledge_to_chapter()` |
| **复习策略推荐** | 推荐高频考点/易错点/压轴题 | Tool Calling：`get_review_recommendation()` |
