**参数面板规则**：
1. **展开位置**：从输入框容器内向上展开，不遮挡对话历史
2. **参数来源**：所有参数统一展示，不区分"已确认"和"AI 默认"
3. **组件类型**：
   - 下拉框：学科、年级、学期、教材、章节、场景
   - 标签多选：题型（点击切换选中/取消）
   - 三段滑块：难度（独立调整简单/中等/困难比例，自动归一化）
   - 范围滑块：题量、时长
   - 文本框：补充说明
4. **实时反馈**：拖动滑块实时显示数值
5. **约束传递**：补充说明传递给 Agent 作为额外约束

**分栏规则**：
- **触发时机**：试卷生成完成后，自动切换为分栏视图
- **左栏**：对话历史 + 生成统计 + 操作按钮，可折叠/展开
- **右栏**：试卷排版预览（主区域），分题型展示
- **返回对话**：点击"← 返回对话"收起右栏，回到纯对话视图
- **宽度调整**：左栏可拖拽调整宽度（20%-40%），或通过折叠按钮完全收起

**右栏 - 试卷预览详细说明**：
- 每题显示：题号、内容、选项（选择题）、分值
- 答案默认隐藏，点击"显示答案"全局切换
- 解析默认折叠，点击展开
- Hover 时显示操作按钮：
  - 🔄 **换一题**：重新生成该题
  - ✏️ **改编**：弹出改编面板
  - 🗑️ **删除**：删除该题

**改编方向说明**：
- **提高难度**：增加计算复杂度、综合考查多个知识点
- **降低难度**：简化条件、减少计算步骤
- **转换题型**：选择题→填空题→解答题
- **换考查点**：保持难度和题型，更换考查的知识点
- **换情境**：更换题目背景（如生活场景、几何图形等）
- **简化/扩展**：增加或减少题目步骤

## 2. 页面布局

### 2.1 路由结构

| 路由 | 页面 | 说明 |
|-----|------|------|
| `/` | 首页 | 场景选择 + 需求输入入口 |
| `/chat/:sessionId` | 对话页 | AI 多轮对话 + 参数确认 + 试卷预览 |

**注意**：试卷预览不是独立页面，而是对话页的分栏视图。

### 2.2 响应式布局

| 屏幕尺寸 | 布局适配 |
|---------|---------|
| **桌面端（≥1280px）** | 左右分栏，左栏 30%，右栏 70% |
| **平板端（768-1279px）** | 左右分栏，左栏可折叠 |
| **移动端（<768px）** | 上下布局，对话历史可抽屉展开 |

## 4. 交互规则

### 4.1 参数面板交互

| 交互 | 行为 |
|-----|------|
| **首次展开** | 从输入框容器向上展开，带淡入动画 |
| **修改参数** | 实时更新状态，不立即触发生成 |
| **拖动滑块** | 实时显示数值，松开鼠标后更新状态 |
| **难度分布** | 三段滑块自动归一化（总和 100%） |
| **题型多选** | 点击标签切换选中状态（选中高亮） |
| **确认生成** | 验证必填参数，通过后触发 Agent 生成 |
| **关闭面板** | 点击面板外区域或按 ESC 键关闭 |

### 4.2 试卷预览交互

| 交互 | 行为 |
|-----|------|
| **题目 Hover** | 显示操作按钮（重新生成、改编、删除） |
| **答案显隐** | 全局切换，影响所有题目 |
| **解析展开** | 单题独立控制，点击"解析 ▼"展开/收起 |
| **拖拽排序** | 拖动题目卡片调整顺序，支持跨题型拖拽 |
| **重新生成** | 弹出确认框，确认后替换该题 |
| **改编题目** | 弹出改编面板（见 1.2 ⑦） |
| **删除题目** | 弹出确认框，确认后删除并重新计算题号 |

### 4.3 拖拽排序规则

**使用库**：`@dnd-kit/core`

**拖拽行为**：
- 拖动题目卡片时，其他题目自动让位
- 支持跨题型拖拽（如选择题拖到填空题组）
- 拖拽结束后自动重新计算题号
- 拖拽过程中显示插入位置的视觉提示（蓝色横线）

**约束**：
- 子题（如阅读理解的小题）只能在父题内拖拽
- 不允许将父题拖入子题
