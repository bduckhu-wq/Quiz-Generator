# 相似题生成技能

## Overview

基于原题生成保持核心学习目标但表层维度变化的相似题。通过四层可控模型精确控制相似度，统一采用70%相似度设计。

**核心理念：相似度是比率，题目体量越大，维持同样相似比需要改变的部分越多。**

## Four-Layer Control Model

### 核心层（锁死 = 0%变化）

| 变量 | 说明 |
|------|------|
| 最小知识点 | 精准到考点，如「勾股数定义」 |
| 核心数学模型 | 公式、等量关系、几何拓扑 |
| 解题逻辑链 | 步骤顺序、突破口 |
| 能力考查层级 | 理解/应用/综合 |
| 设问核心目标 | 列方程/求值/判对错 |

**一动就不是相似题，是改编题。**

### 结构层（≤10%变化）

| 维度 | 说明 |
|------|------|
| 题型 | 客观题/主观题/实操题 |
| 干扰项类型 | 概念混淆/计算错误/逻辑颠倒/范围遗漏（必须来自真实误解） |
| 求解对象 | 基础量/中间量/参数/关系判定 |
| 设问角度 | 正向/反向/逆向验证/存在性/最值 |
| 设问层级 | 单问/两问递进/多问阶梯/开放探究 |
| 条件呈现 | 显性/间接推导/图表/隐藏条件 |

### 表达层（30-50%变化）

| 维度 | 说明 |
|------|------|
| 背景/场景/人物 | 三要素绑定变化，只记1处 |
| 信息载体 | 纯文字/几何图/函数图像/表格/图文混合 |
| 语言风格 | 严谨书面/生活口语/故事风 |
| 信息顺序 | 顺叙/问题前置/条件拆分 |

### 数值层（80-100%变化）

| 维度 | 说明 |
|------|------|
| 具体数值 | 换数字，可改变整除性 |
| 数量级 | 百/千/万，可大幅改动 |
| 单位 | 涉及单位换算才算变化 |

## 70%相似度规则

| 维度 | 要求 |
|------|------|
| 题型 | 与原题一致 |
| 场景 | 完全陌生化 |
| 结构层 | **至少改变1处** |
| 选项结构 | 必须变化 |
| 干扰项 | 来自模型的不同误解角度 |

**核心原则：场景陌生化 + 结构层变化 + 干扰项真实**

## 体量与变化量

| 体量 | 问数 | 变化要求 |
|------|------|----------|
| 小题 | 1问 | 改2处（跨2维度） |
| 少问题 | 2-3问 | 改3-4处（跨3维度） |
| 中等题 | 3-5问 | 改4-6处（跨3维度） |
| 大题 | 5问以上 | 改6处以上（3维度全动） |

**核心层永远不动。**

## 干扰项设计原则

**所有干扰项必须来自对核心模型的真实误解，禁止硬塞无关约束。**

| ✅ 正确 | ❌ 错误 |
|--------|--------|
| 来自模型的真实误解 | 硬塞无关约束 |
| 四选项对应真实错误类型 | 加无关附加条件 |

## 输出格式

```json
{
  "question": "题目内容，含选项",
  "answer": "正确答案",
  "explanation": "详细解析"
}
```

## 常见错误

| 错误 | 修正 |
|------|------|
| 只改数值和人物名称 | 必须同时改变结构层至少1处 |
| 场景未陌生化 | 场景必须完全陌生化 |
| 干扰项硬塞无关约束 | 必须来自真实误解 |
| 生成不可作答的题目 | 检查条件充分性、计算量合理性、答案唯一性 |
