# 相似题生成技能

## Overview

基于原题生成70%相似度的相似题。**必须严格遵循四层可控模型，禁止跳过任何步骤。**

## 强制执行规则

**必须按顺序执行以下步骤，禁止跳过或合并：**

1. **必须**分析原题四层变量（核心层/结构层/表达层/数值层）
2. **必须**判断题目体量（小题/少问题/中等题/大题）
3. **必须**根据体量确定变化维度数量
4. **必须**实现场景完全陌生化
5. **必须**改变结构层至少1处
6. **必须**验证干扰项来自真实误解
7. **必须**检查题目可作答性
8. **必须**输出JSON格式

## 四层可控模型

### 核心层（禁止改变）

**以下变量禁止修改，修改即为改编题而非相似题：**
- 最小知识点
- 核心数学模型
- 解题逻辑链
- 能力考查层级
- 设问核心目标

### 结构层（必须改变至少1处）

**必须从以下维度选择至少1处进行改变：**
- 题型（客观题/主观题/实操题）
- 干扰项类型（概念混淆/计算错误/逻辑颠倒/范围遗漏）
- 求解对象（基础量/中间量/参数/关系判定）
- 设问角度（正向/反向/逆向验证/存在性/最值）
- 设问层级（单问/两问递进/多问阶梯/开放探究）
- 条件呈现形式（显性/间接推导/图表/隐藏条件）

### 表达层（必须改变）

**必须改变以下至少1项：**
- 背景/场景/人物（三要素绑定变化，只记1处）
- 信息载体（纯文字/几何图/函数图像/表格）
- 语言风格（严谨书面/生活口语/故事风）
- 信息顺序（顺叙/问题前置/条件拆分）

### 数值层（必须改变）

**必须改变以下至少1项：**
- 具体数值
- 数量级
- 单位（涉及换算才算）

## 体量变化规则（强制执行）

| 体量 | 问数 | 必须改变维度数 |
|------|------|----------------|
| 小题 | 1问 | 至少2处（跨2维度） |
| 少问题 | 2-3问 | 至少3-4处（跨3维度） |
| 中等题 | 3-5问 | 至少4-6处（跨3维度） |
| 大题 | 5问以上 | 至少6处（3维度全动） |

## 干扰项设计（强制规则）

**禁止：**
- 硬塞与核心模型无关的约束
- 添加无关附加条件
- 使用随机数值作为干扰项

**必须：**
- 所有干扰项来自对核心模型的真实误解
- 四个选项对应四种真实错误类型
- 干扰项对应学生实际会犯的错误

## 输出格式（强制）

**必须输出以下JSON格式，禁止输出其他格式：**

```json
{
  "question": "题目内容，含选项",
  "answer": "正确答案",
  "explanation": "详细解析"
}
```

## Red Flags（出现以下情况必须重新生成）

- [ ] 只改了数值和人物名称
- [ ] 场景未完全陌生化（如原题跑步→相似题还是跑步）
- [ ] 结构层未改变任何维度
- [ ] 干扰项包含无关约束
- [ ] 题目条件不足以支撑唯一答案
- [ ] 计算量超出常规考试要求
- [ ] 答案存在歧义
- [ ] 输出格式不是JSON

## 常见偷懒行为（禁止）

| 偷懒行为 | 正确做法 |
|----------|----------|
| 只改数字和名字 | 必须同时改变结构层+表达层+数值层 |
| 场景微调（小明→小红） | 场景必须完全陌生化（跑步→航海） |
| 保持原题选项结构 | 选项顺序/正确项位置必须变化 |
| 跳过体量判断 | 必须先判断体量再确定变化量 |
| 输出纯文本 | 必须输出JSON格式 |
